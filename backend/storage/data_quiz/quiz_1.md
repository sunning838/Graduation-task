# 과목: 응용 SW 기초 기술 활용 # 키워드: 정렬 알고리즘, 선택 정렬(Selection Sort)

문제 : 다음 자료에 대하여 “Selection Sort”를 사용하여 오름차순으로 정렬한 경우 PASS 3의 결과는?
> 초기 상태 : {8, 3, 4, 9, 7}

     1) 3, 4, 7, 9, 8
     2) 3, 4, 8, 9, 7
     3) 3, 8, 4, 9, 7
     4) 3, 4, 7, 8, 9

- 해설: **선택 정렬(Selection Sort)**은 해당 회차(PASS)의 자리에 올 가장 작은 값을 찾아 교환하는 방식입니다.
    * **초기 상태**: {8, 3, 4, 9, 7}
    * **PASS 1**: 전체 중 가장 작은 **3**을 첫 번째(8)와 교환 → {**3**, 8, 4, 9, 7}
    * **PASS 2**: 두 번째부터 끝까지 중 가장 작은 **4**를 두 번째(8)와 교환 → {3, **4**, 8, 9, 7}
    * **PASS 3**: 세 번째부터 끝까지 중 가장 작은 **7**을 세 번째(8)와 교환 → {3, 4, **7**, 9, 8}
- 정답: **1) 3, 4, 7, 9, 8**

문제 : 다음 전위식(prefix) `- / * A + B C D E`를 후위식(postfix)으로 옳게 표현한 것은?
> 전위식 : - / * A + B C D E

     1) A B C + D / * E -
     2) A B * C D / + E -
     3) A B * C + D / E -
     4) A B C + * D / E -

- 해설: 전위식을 후위식으로 변환할 때는 연산자를 피연산자 뒤로 보내는 과정을 거칩니다.
    1. `+ B C` → `(B C +)`
    2. `* A (B C +)` → `(A B C + *)`
    3. `/ (A B C + *) D` → `(A B C + * D /)`
    4. `- (A B C + * D /) E` → `A B C + * D / E -`
- 정답: **4) A B C + * D / E -**

- 해설: **BCNF(Boyce-Codd Normal Form)**는 모든 결정자가 후보 키여야 한다는 조건을 만족해야 합니다. 따라서 **결정자이면서 후보 키가 아닌 함수적 종속성을 제거**해야 BCNF가 됩니다.
    * 1번은 3NF, 2번은 2NF, 3번은 4NF의 조건입니다.
- 정답: **4) 결정자이면서 후보 키가 아닌 것 제거**

# 과목: 소프트웨어 개발 보안 구축 # 키워드: 공개키 암호화(Asymmetric Key), RSA

     1) 10명이 공개키 암호를 사용할 경우 5개의 키가 필요하다.
     2) 복호화키는 비공개 되어 있다.

- 정답: **1) 10명이 공개키 암호를 사용할 경우 5개의 키가 필요하다.**
